# Zuno - Task Management Platform

Zuno is a hassle-free task management platform designed for speed and simplicity. It aims to streamline your workflow with an intuitive, distraction-free interface.

*   **Frontend**: Next.js (React), Tailwind CSS

We are building Zuno with a specific philosophy:

*   **Today-first dashboard**: Focus on what needs to happen *now*.
*   **Extremely fast task capture**: Adding a task should be instant.
*   **Three simple statuses only**: Todo, Doing, Done. No complex workflows.
*   **Minimal setup for teams**: Onboard your team in seconds.
*   **Smart automation**: Automation over manual management.
*   **Affordable pricing**: Built for small teams.
